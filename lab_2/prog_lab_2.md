### Задача 1
#### Постановка задачи
Сложение двух целых чисел через указатели.
#### Список идентификаторов
| имя переменной | тип данных | описание                  |
| -------------- | ---------- | ------------------------- |
| x              | int        | первое число              |
| y              | int        | второе число              |
| a              | int        | указатель на первое число |
| b              | int        | указатель на второе число |
#### Код программы
```c
#include <stdio.h>

int main() {
  int x, y;
  int *a = &x, *b = &y;

  printf("%d", *a + *b);
  return 0;
}
```

```

```

### Задача 4
#### Постановка задачи
Выделить память под динамический массив и пройти его указателем. Вывести сумму элементов.
#### Список идентификаторов
| имя переменной | тип данных | описание                             |
| -------------- | ---------- | ------------------------------------ |
| n              | int        | размер вводимого массива             |
| arr            | int        | указатель на массив                  |
| sum            | int        | сумма элементов массива              |
| ptr            | int        | указатель на текущий элемент массива |
#### Код программы
```c
#include <stdio.h>
#include <stdlib.h>

int main() {
  int n;
  scanf("%d", &n);

  int *arr = (int *)malloc(n * sizeof(int));

  for (int i = 0; i < n; i++) {
    scanf("%d", arr + i);
  }

  int sum = 0;
  int *ptr = arr;
  for (int i = 0; i < n; i++) {
    sum += *ptr;
    ptr++;
  }

  printf("%d", sum);
  free(arr);

  return 0;
}
```
#### Результаты работы программы
```
4
2 3 4 5
14
```

### Задача 5
#### Постановка задачи
Выделить память под динамический массив и пройти его указателем. Вывести элементы в обратном порядке.
#### Список идентификаторов
| имя переменной | тип данных | описание                             |
| -------------- | ---------- | ------------------------------------ |
| n              | int        | размер вводимого массива             |
| arr            | int        | указатель на массив                  |
| ptr            | int        | указатель на текущий элемент массива |
#### Код программы
```c
#include <stdio.h>
#include <stdlib.h>

int main() {
  int n;
  scanf("%d", &n);

  int *arr = (int *)malloc(n * sizeof(int));

  for (int i = 0; i < n; i++) {
    scanf("%d", arr + i);
  }

  int *ptr = arr + n - 1;
  for (int i = 0; i < n; i++) {
    printf("%d ", *ptr);
    ptr--;
  }

  free(arr);

  return 0;
}
```
#### Результаты работы программы
```
3
10 20 30
30 20 10
```

### Задача 6
#### Постановка задачи
Вывести значения байтов переменной в десятичном виде используя unsigned char \*.
#### Список идентификаторов
| имя переменной | тип данных    | описание                |
| -------------- | ------------- | ----------------------- |
| x              | int           | вводимое число          |
| ptr            | unsigned char | указатель на байт числа |
#### Код программы
```c
#include <stdio.h>

int main() {
  int x;
  scanf("%d", &x);

  unsigned char *ptr = (unsigned char *)&x;

  for (size_t i = 0; i < sizeof(int); i++) {
    printf("%zu: %u\n", i, ptr[i]);
  }

  return 0;
}
```


### Задача 7
#### Постановка задачи
Выделить память под матрицу 2\*3 (ввод элементов в строке) как массив указателей, вывести построчно, освободить.
#### Список идентификаторов
| имя переменной | тип данных | описание                  |
| -------------- | ---------- | ------------------------- |
| matrix         | int        | указатель на матрицу 2\*3 |

#### Код программы
```c
#include <stdio.h>
#include <stdlib.h>

int main() {
  int *matrix[2];
  for (int i = 0; i < 2; i++) {
    matrix[i] = (int *)malloc(3 * sizeof(int));
  }

  for (int i = 0; i < 2; i++) {
    for (int j = 0; j < 3; j++) {
      scanf("%d", &matrix[i][j]);
    }
  }

  for (int i = 0; i < 2; i++) {
    for (int j = 0; j < 3; j++) {
      printf("%d ", matrix[i][j]);
    }
    printf("\n");
  }

  for (int i = 0; i < 2; i++) {
    free(matrix[i]);
  }

  return 0;
}
```
#### Результаты работы программы
```
1 2 3 4 5 6  
1 2 3    
4 5 6
```
